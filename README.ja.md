# HinanMemo
ひなんメモは、ユーザーの現在地周辺の避難所情報を確認できるiOSアプリです。

## 機能
- ユーザーの位置情報を取得し、近隣の避難所を表示
- 避難所の詳細情報（名称、位置情報など）を表示
- 避難所への経路ナビゲーションをGoogle Mapで起動

## 必要環境
- iOS 9.0以降
- Swift 2.2

## 使い方
1. アプリを起動すると、現在地周辺の避難所情報が表示されます。 
2. 一覧から避難所を選択すると、詳細情報と経路ナビゲーションが表示されます。

## データ・API
本アプリでは、オープンデータポータルサイトの防災情報APIを利用しています。
