# HinanMemo
ひなんメモは、iOSアプリ上で災害時の避難所情報を確認できるアプリです。ユーザの現在地周辺の避難所情報を地図上に表示し、簡単に確認できるようになっています。

## デモ
アプリの実際の動作は[Youtube](https://youtu.be/123456789)で確認できます。

## 機能
- ユーザの現在地を取得し、周辺の避難所情報を表示
- 避難所の詳細情報（名称、位置情報など）を表示
- 避難所までのナビゲーションをGoogle Mapで起動

## 必要環境
- iOS 9.0以降
- Swift 2.2

## 使い方
1. アプリをインストールする
2. アプリを起動すると、現在地周辺の避難所情報が表示される
3. 一覧から避難所を選択すると、詳細情報と経路ナビゲーションが表示される

## データ・API
本アプリでは、オープンデータポータルサイトの防災情報APIを利用しています。
